# @thumbsup/theme-classic

One of the built-in themes for https://github.com/thumbsup/thumbsup.

---

## Usage

```bash
thumbsup --theme classic
```

## Screenshots

![albums](docs/albums.png)
![media](docs/media.png)

## Developing

Testing the theme

```bash
thumbsup --input /photos --output ./tmp --theme-path ./theme
```

Publishing the theme:

```bash
npm publish
```
